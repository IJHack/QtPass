Why QtPass?

Built on the proven pass ecosystem with native look and feel on every platform.

🔐

GPG Encryption

Your passwords are encrypted with GPG - the same encryption used by security experts worldwide.

🔄

Git Sync

Your password store syncs via Git. You control where it's hosted. Always have a backup.

📱

Cross-Platform

Works seamlessly on Linux, macOS, Windows, and FreeBSD with native look and feel.

💳

Smartcard Support

Supports YubiKeys and smartcards. Your private keys never leave the device.

🌍

20+ Languages

Available in over 20 languages. Help translate on Weblate.

🍎

No Vendor Lock-in

Works with pass ecosystem. Use QtPass alongside other clients on mobile, CLI, or browser.

Quick Install

Get QtPass installed in minutes on your preferred platform.

🐧 Linux 
# Debian/Ubuntu
apt-get install qtpass

# Arch
pacman -S qtpass

# Fedora
dnf install qtpass
🪟 Windows 
# Via Chocolatey
choco install qtpass

# Or download from GitHub
github.com/IJHack/QtPass/releases
🍎 macOS 
# Via Homebrew
brew install qtpass --cask

# Or download from GitHub
github.com/IJHack/QtPass/releases
