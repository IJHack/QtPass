auto Util::isValidKeyId ( const QString & keyId) -> bool
static

Check if a string looks like a valid GPG key ID. Accepts:

  • Key IDs: 8-40 hex characters (0-9, A-F, a-f), optional 0x prefix
  • Emails: any string containing @ (e.g., user@.nosp@m.doma.nosp@m.in.or.nosp@m.g, user@.nosp@m.doma.nosp@m.in.or.nosp@m.g)
  • Special prefixes: leading @, /, #, or & (any content after prefix)
    Parameters
    keyIdThe string to validate.
    Returns
    true if the key ID format is valid, false otherwise.

Returns a regex matching common remote/network protocol schemes.

Matches http://, https://, ftp://, ftps://, ssh://, sftp://, webdav://, webdavs://
