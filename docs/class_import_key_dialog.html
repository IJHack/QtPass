Detailed Description

Dialog for importing GPG keys from file or clipboard.

Allows users to import GPG public keys without leaving QtPass. Offers two import paths: from file or from clipboard text.

◆ parseGpgImportOutput()

auto ImportKeyDialog::parseGpgImportOutput ( const QString & output) -> QString
static

Parse GPG import output to extract a key id or fingerprint.

Prefers locale-independent status lines ([GNUPG:] IMPORT_OK and [GNUPG:] IMPORTED) when present, then falls back to the human-readable "gpg: key ...: imported" line (English locale only).

Parameters
outputCombined stdout/stderr from gpg --import --status-fd 1.
Returns
The imported key id or fingerprint, or empty if not found.
