QtPass

Introduction

QtPass is a multi-platform GUI for pass, the standard unix password manager.

https://qtpass.org/

Installation

Dependencies

  • QtPass requires Qt 6.x or later (Qt 5.15 retained for legacy support).
  • The Linguist package is required to compile the translations.
  • For use of the fallback icons the SVG library is required.

At runtime the only real dependency is gpg2 but to make the most of it, you'll need git and pass too.

From source

On most *nix systems all you need is:

qmake6 && make && make install (use qmake for Qt5 legacy builds)
