Detailed Description

Represents the parsed contents of a password file.

Splits the file into a password line, named key/value fields, and remaining data. Use FileContent::parse to construct an instance.

◆ parse()

auto FileContent::parse ( const QString & fileContent,
const QStringList & templateFields,
bool allFields ) -> FileContent
static

parse parses the given fileContent in a FileContent object. The password is accessible through getPassword. The named value pairs (name: value) are parsed and depending on the templateFields and allFields parameters accessible through getNamedValues, getRemainingData or getRemainingDataForDisplay.

Parameters
fileContentthe file content to parse.
templateFieldsthe fields in the template. Fields in the template will always be in getNamedValues() at the beginning of the list in the same order.
allFieldswhether all fields should be considered as named values. If set to false only templateFields are returned in getNamedValues().
Returns
A FileContent instance with password, named values, and remaining data populated from the parsed input.
