11 ## Translations
12 
13 * Add you language to the `qtpass.pro` file
14  under TRANSLATIONS and in the `resources.qrc` file.
15 * Next run the command `lupdate qtpass.pro` which will create the localization files.
16 * Edit your file with (let's imagine your language is sv_SE (Swedish)
17  `linguist localization/localization_sv_SE.ts`
18 
19 Qt Linguist has very nice in-context translation options [for translators](https://doc.qt.io/qt-5/linguist-translators.html)
