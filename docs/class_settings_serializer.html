Detailed Description

Maps AppSettings to and from a QSettings store.

Pure, side-effect-free translation between the flat AppSettings value object and the on-disk QSettings keys (the same keys QtPassSettings uses). Unlike the legacy getters it performs no path normalisation, directory creation, screen-centring or .git auto-detection — it only reads and writes raw values. That makes it unit-testable against a throwaway QSettings without any singleton or widget setup.
